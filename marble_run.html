Marble Run 滾珠機械裝置

讓你的視野穿梭——不遠處的海洋、與之相連的天空、壓著我們的大氣層、光年外的星體。一切相互連結，本為一體。只要你用好奇的雙眼，就可以窺探到宇宙的美麗。只要你願意用真誠的心，一步一步踏過光影，就會獲得每件事物蘊含的真理。

這件滾珠機械裝置(Marble Run Machine)，運用了不同領域的知識，製作出各種機關，希望讓你感受到科學的神奇、有趣之處。

最上方的六角形代表「大自然」，左下方的六角形代表「工業」，右下方的六角形代表「科技」。每個六角形「世界」各有特色，各自精彩。 然而，他們並非互相隔絕，「大自然」和「工業」孕育出左上方的木琴，「工業」和「科技」孕育出最下方的儲珠機關，「科技」和「大自然」孕育出右上方的Automata機關。每個「世界」相互連結，孕育出各種美麗。 整個世界就像一個巨型機關，內含各式各樣的奧秘，等待我們去細心發掘。

這個Marble Run Project一點都不簡單，感謝樂在製造的信任，讓我實現這件裝置。
