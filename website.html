語音識別系統的演示網站。

整個網站只用了HTML,CSS和Javascript。要求是，介面乾淨，可offline使用。

這個網站也是自己寫的 :D

This website was created as part of the CUHK SEEM Dysarthric Speech Recognition Research Project.
It was built from scratch using HTML, CSS and Javascript, without external dependencies.
The aim was to create a website that could demonstrate the speech recognition system clearly, while being light-weight and available for offline viewing.

Of course, the website that you are reading was coded from sractch as well :D
